# Remembering Edley Gibbons

A memorial website created in loving memory of my grandfather, **Edley Sylvester Gibbons**.

The website was made for my dad and our family as a place to remember his life, share loving words and keep his memory alive.

## Live Website

[Visit the memorial website](https://tribute-website-qnrdep77z-ohnara1s-projects.vercel.app)

## About the Website

The site includes:

- A memorial hero section dedicated to Edley
- A collection of family tributes and loving messages
- A tribute form where visitors can submit their own memories and messages
- A responsive design for desktop, tablet and mobile
- A custom loading screen
- Smooth scrolling and simple animations
- A warm gold, cream and dark brown colour scheme inspired by remembrance and family

## Tribute Form

Visitors can use the **Leave a Tribute** button to open a form and submit:

- Full name
- Email address
- A memory or tribute

Form submissions are handled using **Formspree**.

## Built With

- HTML
- CSS
- JavaScript
- Formspree
- Vercel

## Purpose

This project is personal to me and my family. I wanted to create something meaningful that could preserve memories of my grandfather and give family members a place to share their own words about him.

It has also been a project for me to continue improving my front-end development skills, particularly responsive CSS, layouts, animations and JavaScript DOM interaction.
